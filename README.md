# Lab 1: LLM API Foundation

## Overview
This repository contains the completed assignments for Lab 1 of the VinAI AI Practical Competency Program. The goal of this lab is to explore LLM APIs, implement various functions to call models, compare performance/cost, and build a streaming chatbot.

## Repository Structure
- `solution.py`: Main implementation of LLM API functions.
- `exercises.md`: Detailed reflection and analysis of experiments.
- `requirements.txt`: Project dependencies.
- `tests/`: Automated test suite to verify the implementation.

## Getting Started

### Prerequisites
- Python 3.10+
- An OpenAI API Key (optional for manual testing, but tests use mocks).

### Installation
1. Clone this repository.
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

### Running Tests
To verify the implementation, run:
```bash
pytest tests/ -v
```
